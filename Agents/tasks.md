## Conventions

- Task IDs: `LAYER-NNN` e.g. `FE-001`, `BE-012`, `QA-003`, `OPS-001`
- One agent owns one task at a time — no shared ownership
- Blocked tasks must name what they are waiting on (another task ID or decision)
- project-orchestrator reviews this file at the start of every session
